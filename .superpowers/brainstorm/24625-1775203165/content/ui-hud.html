Key UI Decisions

  • HUD is HTML/CSS overlay — not rendered in 3D. Simpler, more responsive, easier to style.
  • HP bar is in 3D — floats above player model using R3F's Html component or a 3D sprite.
  • Cook button grays out until both cauldron slots are filled, then glows amber.
  • Recipe preview — when both slots are full, show the resulting spell name before cooking (e.g., "= Inferno 🔥").
  • Card played animation — card slides from hand to cauldron slot. New card fades in.
  • Death sequence — slo-mo (timeScale 0.2) → desaturate scene → fade in The Final Bill.
